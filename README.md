# Telegram ChatGPT Bot

## Описание

Это Telegram-бот, который интегрируется с OpenAI (через китайского вендора) для генерации ответов на сообщения пользователя. Бот поддерживает многократные диалоги, сохраняет контекст и позволяет сбрасывать историю.

## Возможности

* Обработка команд `/start` и `/help`
* Ответ на любые текстовые сообщения через ChatGPT API
* Сохранение истории сообщений для многократного диалога
* Сброс контекста с помощью команды `/start` или кнопки "Новый запрос"

## Требования

* Python 3.12
* Библиотеки:

  * `python-telegram-bot`
  * `openai`

## Установка

1. Клонируйте репозиторий:

```bash
git clone https://github.com/force278/zabota.git
cd zabota
```

2. Создайте виртуальное окружение и активируйте его:

```bash
python3 -m venv venv
source venv/bin/activate
```

3. Установите зависимости:

```bash
pip install python-telegram-bot openai
```

## Настройка

1. Создайте файл `.env` или экспортируйте переменные окружения:

```bash
export TELEGRAM_TOKEN="ваш_токен_бота"
export OPENAI_API_KEY="ваш_api_key"
export LAOZHANG_BASE_URL="https://api.laozhang.ai/v1"
```

## Запуск бота

```bash
python3 main.py
```

## Использование

* Отправьте `/start` чтобы начать диалог и очистить контекст
* Отправьте любое текстовое сообщение, чтобы получить ответ от модели
* Нажмите кнопку "Новый запрос" для сброса истории
* Используйте `/help` для справки
